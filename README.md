# Monthly

A Malaysian* budgeting app I personally use.

I used to track my budget in Notepad — the math was tedious, even though I could've just asked AI. So I built an app instead... then kept adding features along the way.

## The Idea

Tracking daily spending is a hassle. I've always preferred a different approach: **allocate the money first, then spend.**

> *"Don't start the day until you have it finished.*
> *Don't start the week until you have it finished.*
> *Don't start the month until you have it finished. Plan your day."*
>
> — Jim Rohn

## Features

- **Malaysian payroll calculator** — automatically computes EPF, SOCSO, and EIS deductions from your gross salary
- **Monthly Commitments** — organise your fixed expenses into collapsible, drag-to-reorder categories
- **Surplus / Deficit tracker** — instantly see what's left after all commitments
- **Uncommitted & Breakdown tables** — plan where the remaining money goes
- **Save & restore** — data persists in your browser via localStorage
- **Export / Import** — back up and restore your budget as a JSON file
- **Drag & drop import** — drop a JSON file directly onto the page

*Will probably standardize it for global/general use later.
